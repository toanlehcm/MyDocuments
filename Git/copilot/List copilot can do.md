Tôi có thể giúp bạn với nhiều tác vụ lập trình trong VS Code. Dưới đây là danh sách cụ thể:

## 🔍 Khám phá & Phân tích Code
- Giải thích code (cách hoạt động, logic, các khái niệm)
- Tìm kiếm và phân tích codebase
- Review code và đưa ra đề xuất cải thiện
- Phát hiện bugs và vấn đề tiềm ẩn

## ✏️ Viết & Chỉnh sửa Code
- Viết code mới (functions, classes, modules)
- Refactor code hiện tại
- Thêm features mới
- Sửa lỗi (bugs, syntax errors, logic errors)
- Tối ưu hiệu suất

## 🧪 Testing
- Viết unit tests
- Viết integration tests
- Tạo test cases và scenarios
- Debug tests

## 📝 Documentation
- Viết comments và docstrings
- Tạo README files
- Viết technical documentation
- Giải thích API endpoints

## 🔧 Công cụ đặc biệt
- **Explore Agent**: Khám phá codebase nhanh chóng
- **Agent Customization Skill**: Tạo/chỉnh sửa custom instructions, skills, agents cho VS Code

## 💡 Tư vấn
- Best practices
- Design patterns
- Architecture decisions
- Library/framework recommendations

## 📦 Các ngôn ngữ & công nghệ
- JavaScript/TypeScript, Python, Java, C#, Go, Rust, PHP...
- React, Vue, Angular, Node.js...
- SQL, MongoDB, PostgreSQL...
- Docker, Git, CI/CD...
