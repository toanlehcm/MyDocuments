## ✔ Cách Duplicate file về workspace bạn đang dùng

1. Mở link file (bằng account tai.lt)
2. Trong Figma chọn:

```
Menu → Duplicate to your Drafts
```

hoặc

```
File → Duplicate
```

→ File sẽ vào Drafts (hoặc chọn workspace tai.lt’s team)

## 🛠 Lưu ý khi dùng với MCP

Sau khi duplicate:

✔ Chắc chắn file nằm trong workspace -> tai.lt
✔ Bạn có quyền Editor
✔ Bật Dev Mode nếu có seat
✔ Tạo Personal Access Token
✔ Dùng link dạng:

```
https://www.figma.com/file/{FILE_KEY}?node-id={NODE_ID}
```
