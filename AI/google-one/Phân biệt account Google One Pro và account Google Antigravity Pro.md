Hai loại này **rất dễ nhầm** vì đều do Google cung cấp nhưng **hoàn toàn khác sản phẩm**.
Mình giải thích rõ theo **Product → Mục đích → Cách kiểm tra**.

---

# 1️⃣ Google One Pro (Google AI Pro)

### Bản chất

Đây là **gói subscription của Google One**.

👉 Mục đích chính:

- Cloud storage
- AI Gemini
- NotebookLM
- AI video
- Google ecosystem

### Ví dụ gói

| Plan               | Storage  | AI                        |
| ------------------ | -------- | ------------------------- |
| Google One Basic   | 100 GB   | Gemini basic              |
| Google One Premium | 2 TB     | Gemini advanced           |
| **Google AI Pro**  | **2 TB** | **Gemini Pro + AI tools** |

### Tính năng

- 2 TB Google Drive
- Gemini Advanced
- NotebookLM Pro
- AI video generation
- Gmail / Docs AI
- Family sharing

👉 Dashboard quản lý

**Google One**

[https://one.google.com/](https://one.google.com/)

# 2️⃣ Google Antigravity Pro

### Bản chất

Đây là **AI developer platform**.

Nó giống như:

- Copilot
- Cursor
- Claude Code
- AI coding agent

### Mục tiêu

**AI for coding**

Có:

- Antigravity IDE
- AI coding agents
- Antigravity cockpit
- token usage
- AI model access

---

### Các model thường dùng

Ví dụ:

- Gemini
- Claude
- GPT
- DeepSeek

---

### Tính năng

| Feature         | Description      |
| --------------- | ---------------- |
| AI coding agent | viết code        |
| debugging       | fix bug          |
| code generation | generate project |
| terminal agent  | chạy command     |
| cockpit         | theo dõi token   |
