## 🐛 STEPS TO REPRODUCE
1. Vào RCM → Invoice tab
2. Click "+ Add more view" → check view "Outstanding"
3. Hover vào tag "Outstanding" → click "x" để xóa khỏi Quick View list
4. Confirm bước 6: "Outstanding" bị uncheck trong dropdown → Passed
5. **Refresh trang**
6. **Bug xảy ra:** Outstanding xuất hiện lại trong Quick View list và bị auto-checked

**Expected:** Sau refresh, Outstanding vẫn bị uncheck và không xuất hiện trong list.

## 🔍 ROOT CAUSE (đã xác định)

**File:** `Client/app/scripts/controllers/insurance/invoicePtE.js`

Khi user xóa quick view bằng nút "x" (icon tag), function `_deleteQuickView()` (line 3410) được gọi:
- Xóa view khỏi `$scope.quickView.quickViewList` (UI in-memory)
- Set `v.selected = false` trong `$scope.quickView.allViewList` (UI in-memory)
- **KHÔNG gọi API** `v4AddRemoveQuickView` để persist thay đổi lên server

Khi refresh page, `initializer()` (line 548) gọi `_getDataViewAndUserPreference()` — lấy dữ liệu từ server — trong đó `_res.QuickViewList` vẫn còn "Outstanding" (vì chưa được xóa phía server).

### So sánh 2 cách xóa:

| Cách | Function | Gọi API persist? |
|------|----------|-----------------|
| Bỏ check trong "+ Add more view" | `_addMoreView(view)` (line 3449) | ✅ Có — `_callAPIAddRemoveQuickView("remove", id)` |
| Click "x" trên tag Quick View | `_deleteQuickView(view)` (line 3410) | ❌ KHÔNG — chỉ xóa khỏi UI |

---

## ✅ SOLUTION (chưa implement)

Thêm vào `_deleteQuickView()` lời gọi `_callAPIAddRemoveQuickView("remove", view._id)` trước khi xóa UI, hoặc sau khi xóa (optimistic update).

Pattern cần theo (lấy từ `_addMoreView`):
```javascript
function _deleteQuickView(view) {
    if (!view) return;
    // Bước 1: Persist lên server TRƯỚC
    _callAPIAddRemoveQuickView("remove", view._id).then((res) => {
        if (!res) return;
        // Bước 2: Sau khi server OK, mới update UI (đang làm ngược)
        allViewList = res || [];
        allViewList.forEach((v) => (v.selected = v.Selected));
        $scope.quickView.allViewList = _orderAllView(allViewList);
        quickViewList = _.orderBy(allViewList.filter((i) => i.Selected) || [], ["Order"], ["asc"]);
        $scope.quickView.quickViewList = _.cloneDeep(quickViewList);
        // Handle selected view reset nếu cần
        if ($scope.quickView.selectedQuickView === view._id) {
            $scope.quickView.selectedQuickView = defaultViewOfSystem._id;
            _renderViewToPanelFilter(defaultViewOfSystem);
            _applyView(false);
        }
    });
}
```
