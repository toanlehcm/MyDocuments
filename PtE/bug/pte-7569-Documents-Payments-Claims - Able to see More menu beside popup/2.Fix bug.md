## #1: Chuỗi sự kiện khi `$('#btn-more').click()` được gọi

```
$('#btn-more').click()
│
├─► jQuery dispatches synthetic click event lên element #btn-more
│
├─► Bootstrap's dropdown plugin handler xử lý (được bind via event delegation)
│   Bootstrap kiểm tra: parent <div class="dropdown"> có .show không?
│   → CÓ .show → gọi .dropdown('hide')
│   → Bootstrap removes class .show khỏi parent <div class="dropdown">
│
├─► CSS transition kích hoạt:
│   .show>.dropdown-menu__slow-hidden  ← không còn match nữa
│   → opacity: 100% → 0%  (transition 0.1s)
│   → max-height: 100vh → 0  (transition 0.3s)
│   → max-width: 100vw → 0   (transition 0.3s)
│   → Menu biến mất trong 0.3s
│
└─► AngularJS ng-click cũng fires → onClickMoreActions() chạy
    (tải task count — side effect vô hại)
```
