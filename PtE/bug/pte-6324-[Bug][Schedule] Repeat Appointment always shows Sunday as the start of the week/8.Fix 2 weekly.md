## 🔍 Root Cause thực sự

**`clientDoc.DateStart` là STRING** (`"Monday"`, `"Sunday"`), **KHÔNG phải NUMBER**.

```javascript
// Schema (v2_Client.js dòng 50):
DateStart: String,

// Cách dùng khắp codebase:
var dateStart = clientData.DateStart ? clientData.DateStart : "Sunday"
// → Giá trị: "Sunday", "Monday", v.v.
```

Fix trước đó dùng `clientDoc.DateStart` trực tiếp làm số → `(4 - "Monday" + 7) % 7 = NaN` → `subtract(NaN)` → moment giữ nguyên ngày gốc → window bắt đầu từ start date → sinh 14 ngày liên tiếp.

### Fix mới: thêm `_dayNameToNumber` map

```javascript
var _dayNameToNumber = { 'Sunday': 0, 'Monday': 1, 'Tuesday': 2, 
                         'Wednesday': 3, 'Thursday': 4, 'Friday': 5, 'Saturday': 6 }
var _weekStartDay = _dayNameToNumber[clientDoc.DateStart] !== undefined 
    ? _dayNameToNumber[clientDoc.DateStart] : 0
```
