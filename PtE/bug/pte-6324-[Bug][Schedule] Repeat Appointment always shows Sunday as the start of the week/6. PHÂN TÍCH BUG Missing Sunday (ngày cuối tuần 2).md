## 🔍 PHÂN TÍCH BUG: Missing Sunday (ngày cuối tuần 2)

**Dựa trên ảnh chụp:**
- Start Date: Monday 04/13/2026
- 7 ngày/tuần, Weekly, 2 Weeks
- Nhận được 13 ngày (Mon 04/13 → Sat 04/25), **thiếu Sun 04/26**

---

**File:** `Server/logicMongo/utils/AppointmentUtil.js`
**Dòng:** 997–998

### 📋 Cơ chế lỗi — Giải thích chính xác

```
Clinic cấu hình: Mon = ngày đầu tuần, Sun = ngày cuối tuần

moment.js DEFAULT locale: Sun = day 0 (đầu tuần), Sat = day 6 (cuối tuần)
                          (US locale, không phải ISO)

appointment.StartDateTime = Monday, April 13, 2026

startOf('week') → Sun April 12  ← SAI (lẽ ra Mon Apr 13)
endOf('week')   → Sat April 18  ← SAI (lẽ ra Sun Apr 19)

While loop chạy: [Sun Apr 12 → Sat Apr 18]
```

**Kết quả vòng lặp với repeatNumber=2:**

| Ngày trong window | i=0 (tuần 1) | i=1 (tuần 2) | Kết quả |
|:---:|:---:|:---:|:---|
| Sun Apr 12 | Apr 12 (filtered, trước startDate) | **Apr 19** | 1 lịch |
| Mon Apr 13 | **Apr 13** ✓ | **Apr 20** ✓ | 2 lịch |
| Tue Apr 14 | **Apr 14** ✓ | **Apr 21** ✓ | 2 lịch |
| Wed Apr 15 | **Apr 15** ✓ | **Apr 22** ✓ | 2 lịch |
| Thu Apr 16 | **Apr 16** ✓ | **Apr 23** ✓ | 2 lịch |
| Fri Apr 17 | **Apr 17** ✓ | **Apr 24** ✓ | 2 lịch |
| Sat Apr 18 | **Apr 18** ✓ | **Apr 25** ✓ | 2 lịch |
| **Sun Apr 19** | **KHÔNG BAO GIỜ ĐƯỢC SINH RA** | **Apr 26 MẤT** | 0 lịch |

```
Tổng: 1 + 2×6 = 13  ✓ (khớp với bug)
Đúng ra phải: 14 (7 × 2 tuần)
Thiếu: Sun Apr 26 (ngày cuối tuần 2 theo cấu hình Mon-Sun)
```

**Tại sao Sun Apr 26 mất?** Vì **Sun Apr 19 nằm NGOÀI maxDate (Sat Apr 18)** → không được đưa vào window → không có cơ hội generate `Apr 19 + 7 days = Apr 26`.

### 🩹 Fix — Dùng `clientDoc.DateStart` thay vì hardcode locale

```javascript
// AppointmentUtil.js dòng 997-998 và v2_AppointmentLogic.js dòng 17988-17989
// TRƯỚC (BUG):
let minDate = moment(appointment.StartDateTime).tz(clientDoc.TimeZoneValue).startOf('week').endOf('day');
let maxDate = minDate.clone().endOf('week').endOf('day');

// SAU (FIX):
// clientDoc.DateStart = 1 (Monday) theo cấu hình clinic
const weekStartDay = (clientDoc.DateStart !== undefined && clientDoc.DateStart !== null)
    ? clientDoc.DateStart  // 0=Sun, 1=Mon, 2=Tue, ...
    : 0;                   // default Sunday nếu chưa cấu hình

let startOfWeek = moment(appointment.StartDateTime).tz(clientDoc.TimeZoneValue);
// Tính ngày đầu tuần theo cấu hình clinic
let currentDay = startOfWeek.day(); // 0=Sun, 1=Mon, ..., 6=Sat
let diff = (currentDay - weekStartDay + 7) % 7;
let minDate = startOfWeek.clone().subtract(diff, 'days').startOf('day').endOf('day');
let maxDate = minDate.clone().add(6, 'days').endOf('day');
```

**Kết quả sau fix** (StartDate = Mon Apr 13, weekStartDay=1):
```
minDate = Mon Apr 13
maxDate = Sun Apr 19

While loop: [Mon Apr 13 → Sun Apr 19]

Mon Apr 13: Apr 13, Apr 20  ✓
...
Sat Apr 18: Apr 18, Apr 25  ✓
Sun Apr 19: Apr 19, Apr 26  ✓ ← đây rồi!

Tổng: 14 ✓
```
