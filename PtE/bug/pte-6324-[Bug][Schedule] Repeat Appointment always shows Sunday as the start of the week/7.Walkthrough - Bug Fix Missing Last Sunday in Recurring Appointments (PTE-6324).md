# Walkthrough - Bug Fix: Missing Last Sunday in Recurring Appointments (PTE-6324)

## Problem Summary
In `v4CreateAppointmentPtE`, when creating recurring appointments (e.g., 7 days a week for 2 weeks), the last Sunday was often missing from the generated list. This occurred because the system was defaulting to the US locale (Sunday as the first day of the week, Saturday as the last), which misaligned with the clinic's configuration (often Monday-first).

## Changes Made

### 💻 Logic Fix
Modified the week start/end calculation to respect the clinic's configured first day of the week (`clientDoc.DateStart`) instead of relying on the browser/system locale.

#### [AppointmentUtil.js](file:///d:/Sources/pteverywhere/Server/logicMongo/utils/AppointmentUtil.js#L1000-1005)
Updated the `minDate` and `maxDate` calculation in `initAppointmentData`.

#### [v2_AppointmentLogic.js](file:///d:/Sources/pteverywhere/Server/logicMongo/v2_AppointmentLogic.js#L17991-17996)
Applied the identical fix to `_validateBookingDataPtE` to ensure consistency between initial generation and validation.

### 📝 Explanatory Comments
Added concise English comments to each line of the new calculation logic to improve maintainability.

```javascript
var _weekStartDay = (clientDoc.DateStart !== undefined && clientDoc.DateStart !== null) ? clientDoc.DateStart : 0 // Get clinic's starting day of week (0=Sun, 1=Mon, etc.)
var _apptRefDay = moment(appointment.StartDateTime).tz(clientDoc.TimeZoneValue) // Reference date with clinic timezone
var _currentDayNo = _apptRefDay.day() // Get day number of the reference date (0=Sun, 1=Mon, ..., 6=Sat)
var _diffToWeekStart = (_currentDayNo - _weekStartDay + 7) % 7 // Calculate days to subtract to reach start of week
let minDate = _apptRefDay.clone().subtract(_diffToWeekStart, 'days').startOf('day').endOf('day') // Set window start to first day of week
let maxDate = minDate.clone().add(6, 'days').endOf('day') // Set window end to exactly 6 days after start (full week)
```

## Verification Results
- **Scenario:** 7 days a week, 2 weeks repeat, Monday start.
- **Expected:** 14 appointments ending on the second Sunday.
- **Result:** Successfully generates all 14 appointments. The `maxDate` now correctly includes the final Sunday of the window.
