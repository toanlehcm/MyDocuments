**Browser cũ (Chrome < 81) — KHÔNG auto-rotate EXIF:**

```
img.src = base64 (ảnh portrait, orientation=6)
  → Browser cũ KHÔNG xoay → img giữ nguyên raw pixel
  → img.width = 4608, img.height = 3456 (landscape — chưa xoay)
  ↓
Solution A check:
  srcOrientation = 6 → vào if (>= 5 && <= 8)
  browserAlreadyRotated = (height > width) = (3456 > 4608) = FALSE
  orientationForTransform = srcOrientation = 6
  ↓
  → canvas swap + ctx.transform xoay 90° → KẾT QUẢ ĐÚNG ✅
```

**Browser mới (Chrome 81+) — CÓ auto-rotate EXIF:**

```
img.src = base64 (ảnh portrait, orientation=6)
  → Browser mới TỰ xoay → img đã portrait
  → img.width = 3456, img.height = 4608 (portrait — đã xoay)
  ↓
Solution A check:
  srcOrientation = 6 → vào if (>= 5 && <= 8)
  browserAlreadyRotated = (height > width) = (4608 > 3456) = TRUE
  orientationForTransform = 1
  ↓
  → default: break → KHÔNG xoay thêm → KẾT QUẢ ĐÚNG ✅
```
