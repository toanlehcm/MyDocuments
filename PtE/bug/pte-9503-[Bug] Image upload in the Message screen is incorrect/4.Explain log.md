### #1. Orientation = 6 có ý nghĩa gì?

Theo chuẩn **EXIF 2.31** (CIPA DC-008-2012), tag `0x0112` (Orientation) có 8 giá trị:

| Value | Ý nghĩa | Minh họa |
|-------|---------|----------|
| 1 | Bình thường (0°) | ↑ |
| 2 | Lật ngang (mirror) | ↑ (mirrored) |
| 3 | Xoay 180° | ↓ |
| 4 | Lật dọc | ↓ (mirrored) |
| 5 | Lật ngang + xoay 270° CW | ← (mirrored) |
| **6** | **Xoay 90° CW (portrait)** | **→** |
| 7 | Lật ngang + xoay 90° CW | → (mirrored) |
| 8 | Xoay 270° CW | ← |

**Orientation = 6 = Điện thoại cầm dọc (portrait), xoay 90° theo chiều kim đồng hồ.**

📎 **Document chính thức:**
- EXIF Spec: https://www.cipa.jp/std/documents/download_e.html?DC-008-Translation-2023-E
- ExifTool reference: https://exiftool.org/TagNames/EXIF.html (tìm tag 0x0112)
- MDN liên quan: https://developer.mozilla.org/en-US/docs/Web/CSS/image-orientation

### #2. Phân tích kết quả debug — BẰNG CHỨNG XÁC THỰC

```
orientation: 6
[resetOrientation] srcOrientation: 6  img.width: 3456  img.height: 4608
```

**Đây là bằng chứng quyết định xác nhận root cause:**

Ảnh sensor gốc từ camera Samsung Galaxy luôn lưu theo chiều landscape (ngang). Khi chụp portrait (dọc), **raw pixel data** thực tế là **4608×3456** (rộng × cao), rồi gắn EXIF tag `orientation=6` để báo rằng "hãy xoay 90° khi hiển thị".

Nhưng log cho thấy `img.width: 3456, img.height: 4608` (cao > rộng = **đã portrait sẵn**). Điều này chứng minh: **Chrome WebView đã tự động xoay ảnh rồi** trước khi `img.onload` fires.

Rồi code chạy vào `if (4 < 6 && 6 < 9)`:
```javascript
canvas.width = height = 4608   // ← swap thành landscape!
canvas.height = width = 3456
ctx.transform(0, 1, -1, 0, 4608, 0)  // ← xoay thêm 90° nữa!
```

→ **Kết quả: ảnh portrait (3456×4608) bị xoay thêm 90° → thành landscape (4608×3456) → SAI**

### ✅ Root Cause (Xác nhận 100%)

**Double rotation**: Chrome WebView (Android 16, Chrome 146) tự động apply EXIF orientation khi load `new Image()`. Hàm `resetOrientation` không biết điều này, vẫn đọc EXIF raw → apply canvas transform lần 2 → ảnh portrait bị xoay thành landscape.
