# 📋 PTE-8162: Plan Fix — Invoice Reset + Apply Column Bug

## Root Cause Recap

Invoice dùng `$watch("searchTerms.columns")` + closure variable `isChangeDisplayColumns` để detect column changes.
Khi Reset, cả `searchTerms.columns` lẫn `oldView.columns` đều update cùng giá trị → `$watch` thấy GIỐNG → `isChangeDisplayColumns = false` → table chỉ dispatch `"resize"` thay vì `"change-number-of-column"` → **column không reflow**.

Claims dùng `cacheHeaderColumnVisible` (snapshot columns đang hiển thị trên table) để so sánh → luôn detect đúng khi columns thay đổi.
