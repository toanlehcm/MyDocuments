## 🔴 $rootScope MẤT KHI F5 — Danh sách theo mức độ quan trọng

### Lưu ý quan trọng

Khi **không nhấn F5** (Logout → Login): Angular SPA chỉ thay route. `app.run()` block sẽ **chạy lại** và gán giá trị default (line 140-280), sau đó `initDataSession()` overwrite bằng dữ liệu user thật. Vì SPA vẫn sống, quá trình này rất nhanh.

Khi **nhấn F5**: Toàn bộ JS heap bị destroy. Browser phải **tải lại tất cả JS files → parse → compile → bootstrap Angular → chạy app.run() → gọi API v3GetDataSession → initDataSession()**. Quá trình này mất 2-5 giây.
