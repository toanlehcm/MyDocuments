# Giải quyết Main Thread Freeze do `md-contact-chips`

Bản thiết kế này đề xuất 2 phương án Workaround ở cấp View/Controller nhằm bypass chi phí Compile & Layout Thrashing khổng lồ của `<md-contact-chips>` khi khởi tạo Right Panel / `$mdDialog`. Tuân thủ nguyên tắc **Bất biến Thư viện Core** (Không can thiệp source Angular Material).

## User Review Required

Vui lòng xem kỹ bảng **Trade-off** dưới đây để chốt phương án trước khi Executor tiến hành viết code.

| Tiêu chí | Phương án 1 (Defer Mounting) | Phương án 2 (Read/Write Switch Pattern) |
| :--- | :--- | :--- |
| **Cơ chế** | Dùng `ng-if` kết hợp `$timeout` để lùi thời điểm gắn DOM của Chips sau khi Panel đã render và transition xong. | Render 1 `div` tĩnh (read-only) hiển thị data. Chỉ mount `md-contact-chips` khi User chủ động click chọn "Edit". |
| **UX (Trải nghiệm)** | User sẽ thấy ô Search Patient "pop-in" (xuất hiện chậm hơn 1 nhịp, khoảng 300-500ms so với toàn bộ Panel). | Cực mượt. Chớp nhoáng. Nhưng User tốn thêm 1 click nếu muốn add/remove patient. |
| **Performance Gain**| Trung bình (Giải phóng Main Thread lúc đầu, nhưng lúc Chips mount vẫn có vi-khựng nhẹ). | Cực cao (O(1) DOM nodes lúc Init). Chi phí render Chips trả sau theo ngụ ý thao tác (Intent-based). |
| **Code Complexity** | Rất thấp. Chỉ thêm 1 flag `$scope.isChipsReady` và 1 block `$timeout`. | Vừa phải. Cần viết thêm 1 block UI HTML cho trạng thái Read-only và handle event toggle hai state. |

---

## Proposed Changes

### Option 1: Defer Mounting (Khuyến nghị nếu cần giữ nguyên Flow hiện tại)

#### [MODIFY] tasksManagementPtE.js
- Thêm biến `$scope.isPatientChipsReady = false;`.
- Bắn `$timeout(() => { $scope.isPatientChipsReady = true; }, 400);` dựa vào thời điểm open Dialog hoặc load xong data list ban đầu.

#### [MODIFY] tasksManagementPtE.html
- Bọc container chứa `md-contact-chips` vào vòng kiểm định thẻ `ng-if="isPatientChipsReady"`.
- (Tùy chọn) Thêm 1 thẻ Skeleton/Spinner mờ đứng chỗ lúc `ng-if` chưa true để tránh nhảy layout (Layout Shift).

***

### Option 2: Read/Write Switch (Khuyến nghị cho tối ưu triệt để Big Data)

#### [MODIFY] tasksManagementPtE.js
- Thêm biến `$scope.isEditingPatients = false;`.
- Hàm `$scope.enablePatientEdit() = { $scope.isEditingPatients = true; }`.

#### [MODIFY] tasksManagementPtE.html
- **Khối Read-Only (`ng-if="!isEditingPatients"`):** Lặp `ng-repeat` đơn giản qua `filterModel.Patients` để in ra các text/avatar (Chỉ tốn O(N) rất ngọn nhẹ). Kèm nút "Edit / Thêm".
- **Khối Edit (`ng-if="isEditingPatients"`):** Chứa `md-contact-chips` nguyên bản.
