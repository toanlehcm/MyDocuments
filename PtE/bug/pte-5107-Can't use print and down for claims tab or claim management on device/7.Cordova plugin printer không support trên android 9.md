## 🔍 Kết quả Research: `cordova-plugin-printer`

### Câu trả lời ngắn gọn: **KHÔNG** hỗ trợ Android 10+

**Phiên bản mới nhất là `0.8.0` (tháng 3/2019)** — plugin này đã **không còn được maintain** từ năm 2019. Kể cả phiên bản mới nhất vẫn dùng Android Support Library cũ (đã bị Google khai tử, thay bằng AndroidX), nên:

| Platform | Hỗ trợ |
|---|---|
| **Android 4.4 → 9** | ✅ Hoạt động |
| **Android 10+** | ❌ Không hỗ trợ (kể cả v0.8.0) |
| **iOS 10+** | ✅ Hoạt động (dùng AirPrint) |

---

## ⚠️ Đánh giá rủi ro cho dự án hiện tại

**iOS:** Flow hiện tại của `_printPdf` trong `commonService.js` là **an toàn** — download file về local trước, rồi mới gọi `$cordovaPrinter.print(fileURI)`. Plugin hỗ trợ iOS 10+ qua AirPrint.

**Android:** Code hiện tại gọi thẳng `$cordovaPrinter.print(url)` — **chỉ chạy được trên Android 4.4–9**. Android 10+ sẽ không hoạt động.

## 💡 Đề xuất (cho anh cân nhắc)

Vì đây là **plugin cũ và đã chết**, em có 2 hướng:

**Hướng 1 (Ngắn hạn - ít risk):** Giữ nguyên, chỉ cần cập nhật comment và bug report cho QC biết rõ giới hạn:
```javascript
// Android: Cordova Plugin Printer v0.7.3/0.8.0 is obsolete.
// Only supports Android 4.4 - 9 (Android < API 29).
// For Android 10+ (API 29+): print is NOT available.
// iOS: Works via AirPrint. File must be downloaded locally first.
if ($rootScope.DeviceType === 'android') {
    $cordovaPrinter.print(pdfUrl);
    return;
}
```

**Hướng 2 (Dài hạn - cần sprint riêng):** Nâng cấp lên Capacitor Printer plugin (`@capgo/capacitor-printer` hoặc `@capawesome-team/capacitor-printer`) — hỗ trợ cả Android 10+ và iOS đầy đủ. Tuy nhiên **đây là một thay đổi lớn**, cần migrate native code.
