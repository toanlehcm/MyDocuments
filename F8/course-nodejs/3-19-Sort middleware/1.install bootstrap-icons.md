Anh đã chạy `npm install bootstrap-icons` và thêm thẻ `<i class="bi bi-arrow-down-up"></i>` vào view là rất chuẩn rồi. Tuy nhiên, để icon hiển thị lên được thì trình duyệt (browser) cần phải đọc được file CSS và Font của Bootstrap Icons từ thư mục `node_modules` (vì code NodeJS chỉ chạy ở server, client không tự đọc được `node_modules`).

Để xử lý việc này một cách gọn gàng và đúng chuẩn ExpressJS, em đã thao tác giúp anh 2 việc sau:

**1. Cấp quyền truy cập (Static Route) trong `src/index.js`**
Em đã thêm một dòng cấu hình để Express cho phép trình duyệt lấy các file tĩnh (css, font) từ thư mục `bootstrap-icons` thông qua đường dẫn ảo `/vendor/...`:
```javascript
// src/index.js (Dòng 17)
app.use(express.static(path.join(__dirname, 'public')));
app.use('/vendor/bootstrap-icons', express.static(path.join(__dirname, '../node_modules/bootstrap-icons'))); // Mới thêm
```

**2. Nhúng CSS vào Layout chính `src/resources/views/layouts/main.hbs`**
Em đã thêm thẻ `<link>` để tải file CSS của Bootstrap Icons vào Layout dùng chung. Nhờ vậy, từ giờ anh muốn dùng icon ở view nào (như `stored-courses.hbs`) cũng đều hiển thị được luôn:
```html
<!-- src/resources/views/layouts/main.hbs (Dòng 8) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/css/bootstrap.min.css" ...>
<link rel="stylesheet" href="/vendor/bootstrap-icons/font/bootstrap-icons.css"> <!-- Mới thêm -->
```
