Chào anh! Em hiểu vấn đề anh đang gặp phải rồi. Lỗi này là một "cú lừa" kinh điển trong JavaScript liên quan đến **Truthy/Falsy value** đấy!

**🔍 NGUYÊN NHÂN CHÍNH KHÔNG ĐỔI TYPE ĐƯỢC:**
Anh nhìn vào URL này nhé: `?_sort&column=name&type=asc`
Khi ExpressJS phân tích URL trên, nó sẽ tạo ra object `req.query` như sau:
```javascript
{
  _sort: '', // Chú ý: _sort không có giá trị nên bằng chuỗi rỗng
  column: 'name',
  type: 'asc'
}
```

Bây giờ anh nhìn lại code trong `SortMiddleware.js`:
```javascript
if(req.query._sort){  // Sai lầm ở đây
    // ...
}
```
Trong JavaScript, chuỗi rỗng `""` được coi là một giá trị **Falsy** (nghĩa là tương đương với `false` khi để trong điều kiện `if`).
👉 Vì `req.query._sort` bằng `""` nên biểu thức `if ("")` sẽ trả về `false`. Khối code bên trong `if` bị bỏ qua hoàn toàn, dẫn đến `res.locals.sort` vẫn giữ nguyên giá trị `default` ban đầu là `{enabled: false, type: 'default'}`.

**🛠️ CÁCH EM ĐÃ SỬA:**
Em đã vào file `SortMiddleware.js` và cập nhật lại câu lệnh `if`. Giải pháp y hệt như lúc nãy em xử lý lỗi cho anh trong Controller:

Thay vì dùng `if(req.query._sort)`, em đổi thành:
```javascript
if(req.query._sort !== undefined)
```

Lúc này, dù `_sort` có là chuỗi rỗng đi nữa thì giá trị `""` vẫn khác `undefined`, nên `if` sẽ trả về `true` và chạy vào trong để `Object.assign` các thuộc tính `column` và `type` đúng như anh mong muốn.
